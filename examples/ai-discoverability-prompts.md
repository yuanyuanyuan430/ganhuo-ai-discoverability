# AI Discoverability Prompts

High-intent prompts for using `ganhuo-ai-discoverability`.

## GitHub Project

```text
请用 ganhuo-ai-discoverability 分析这个 GitHub 项目，找出它在 AI 搜索、AI 推荐、开发者工具推荐和替代品对比场景里的可发现性机会。输出实体地图、问答场景、AI 入口、README 改法、llms.txt 建议和最优先的 5 个动作。
```

```text
Use ganhuo-ai-discoverability to audit this GitHub repo for AI search optimization. I want ChatGPT, Claude, Perplexity, and AI coding assistants to understand what it does, who it is for, why it is different, and when to recommend it.
```

## SaaS Website

```text
请用 ganhuo-ai-discoverability 分析这个 SaaS 官网。重点看 AI 是否能准确理解产品类别、目标用户、核心能力、差异化、竞品对比和推荐场景。最后给出 3-5 个最值得执行的动作。
```

```text
Use ganhuo-ai-discoverability to create an AI Discoverability Brief for this SaaS website. Focus on entity clarity, buyer-intent prompts, llms.txt, Markdown routes, sitemap/index readiness, and citation-ready landing pages.
```

## Content Site

```text
请用 ganhuo-ai-discoverability 分析我的内容站，找出哪些页面更有机会被 AI 引用，哪些页面需要改标题、摘要、结构、证据和内部入口。
```

## Competitor / Alternative Search

```text
请用 ganhuo-ai-discoverability 设计一组“替代品、对比、怎么选、适合谁”的 AI 问答场景，让我的项目更容易出现在用户比较方案的时候。
```

## llms.txt Planning

```text
请用 ganhuo-ai-discoverability 为这个项目规划 llms.txt 和 llms-full.txt。要求包含项目定位、核心页面、GitHub、文档、使用场景、推荐问题和最值得 AI 读取的资料入口。
```

## Citation Readiness

```text
请用 ganhuo-ai-discoverability 检查这些页面是否适合被 AI 摘引。重点看标题、URL、摘要、证据、对比、限制、数据和可复制答案片段。
```
