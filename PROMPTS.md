# 可复制提示词

把下面任意一段丢给 Codex 使用。

## 快速诊断

```text
请用 ganhuo-ai-discoverability 分析我的网站，让 AI 更容易发现、理解和推荐它。输出实体地图、AI 问答场景、入口文件检查、引用准备度和最优先的 5 个动作。
```

## 开源项目推广

```text
请用 ganhuo-ai-discoverability 分析这个 GitHub 项目。目标是让 ChatGPT、Claude、Perplexity 和 AI 编程助手更容易理解它的用途、优势和适用场景，并给出 README、llms.txt、项目页和内容资产的优化建议。
```

## SaaS 官网优化

```text
请用 ganhuo-ai-discoverability 审视这个 SaaS 官网。重点判断：AI 是否能准确理解产品类别、目标用户、核心能力、差异化、竞品对比和推荐场景。最后给出 3-5 个最值得执行的动作。
```

## 个人品牌 / 作品集

```text
请用 ganhuo-ai-discoverability 帮我梳理个人品牌和作品集，让 AI 在介绍我、我的项目和我的专业方向时更准确、更有说服力。输出实体定位、代表作品、可引用页面和下一步内容建议。
```

## 内容站 / 知识库

```text
请用 ganhuo-ai-discoverability 分析我的内容站，找出哪些页面更有机会被 AI 引用，哪些页面需要改标题、摘要、结构、证据和内部入口。
```

## GitHub 项目 README 重构

```text
请用 ganhuo-ai-discoverability 重构这个 GitHub 项目的 README。要求像项目主页一样推广自己：开头抓人、价值明确、适用场景清楚、示例丰富，并补充 llms.txt 和 llms-full.txt。
```
